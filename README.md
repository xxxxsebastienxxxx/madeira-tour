# Madeira Companion v8

Version propre à déposer à la racine du dépôt GitHub Pages.

## Installation sur GitHub

1. Décompresser l'archive.
2. Dans le dépôt GitHub : Add file > Upload files.
3. Déposer tous les fichiers contenus dans le dossier, pas le dossier lui-même.
4. Valider avec Commit changes.
5. Attendre le déploiement GitHub Pages.

Le bas de la page doit afficher « Madeira Companion v8 ».

Le filtre « Tous les types » contient toutes les catégories habituelles, puis
« 🌅 Coucher de soleil ». Cette option affiche uniquement les 9 lieux marqués
pour le coucher de soleil.
